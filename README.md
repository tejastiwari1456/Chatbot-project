# 🤖 Chatbot Project

An interactive chatbot web interface built using **React, JavaScript, HTML, and CSS**.  
This project simulates a real chat environment where users can send messages and receive responses from the chatbot.

The goal of this project was to practice **frontend development, React component structure, and dynamic UI updates**.

---

## 🚀 Features

- Interactive chat interface
- Dark mode toggle
- Responsive design
- Clean and modular React components

---

## 🛠️ Tech Stack

- React
- JavaScript
- HTML
- CSS

## ⚙️ Installation

Clone the repository

'git clone https://github.com/tejastiwari1456/Chatbot-project.git'

Navigate to the project folder

'cd Chatbot-project'

Install dependencies

'npm install'

Run the project

'npm start'
